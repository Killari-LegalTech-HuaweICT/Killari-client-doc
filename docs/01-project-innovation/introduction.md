# Killari — Resumen ejecutivo

Killari es una solución LegalTech que combina una aplicación de escritorio (Electron) con servicios de IA en Huawei Cloud para acelerar y asegurar el análisis forense de expedientes judiciales.

Puntos clave:

- Objetivo: Automatizar la detección de inconsistencias en documentos legales y facilitar la revisión por parte de fiscales y abogados.
- Integraciones: Huawei ModelArts (IA), OBS (almacenamiento), IAM (seguridad).
- Resultado esperado: Reducir tiempo de revisión manual y mejorar la trazabilidad de la evidencia.
