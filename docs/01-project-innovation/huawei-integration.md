# Integración con servicios Huawei

Este documento lista los servicios de Huawei Cloud y ModelArts utilizados en el proyecto Killari.

## Servicios principales

- **ModelArts**: Entrenamiento y despliegue de modelos de IA para detección y clasificación de patrones en evidencia documental.
- **OBS**: Almacenamiento de grandes volúmenes de evidencias (PDFs, imágenes, vídeos) con control de versiones y políticas de retención.
- **ECS**: Máquinas virtuales para procesos batch y tareas de preprocesado.
- **IAM**: Gestión de identidades y permisos.
- **FunctionGraph / Cloud Functions**: Orquestación de pipelines serverless para invocar análisis bajo demanda.
