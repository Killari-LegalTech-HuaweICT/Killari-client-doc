# Guía de instalación y setup

Este documento cubre los prerrequisitos para ejecutar Killari y los pasos básicos de instalación.

## Prerrequisitos

- Node.js >= 16
- npm o yarn
- (Opcional) Python 3.8+ para tareas de IA locales
- Acceso a una cuenta de Huawei Cloud con credenciales para ModelArts y OBS

## Instalación rápida

1. Clonar el repo:

```bash
git clone https://github.com/Killari-LegalTech-HuaweICT/Killari-client-doc.git
cd Killari-client-doc
```

2. Instalar dependencias:

```bash
npm install
```

3. Ejecutar en modo desarrollo:

```bash
npm run start
```

> Añade aquí cualquier script adicional para el entorno de IA (por ejemplo, creación de entornos virtuales, instalación de dependencias Python, etc.).
